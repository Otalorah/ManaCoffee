# Menú PDF

Para mostrar el menú en la página `/menu`, coloca tu archivo PDF aquí con el nombre `menu.pdf`.

## Instrucciones:

1. Coloca tu archivo PDF del menú en esta carpeta
2. Renómbralo a `menu.pdf`
3. El PDF se mostrará automáticamente en la página de menú

## Ruta del archivo:
```
/public/menu.pdf
```

Si el archivo no existe, se mostrará un mensaje placeholder indicando que el menú estará disponible pronto.
